# Onlyfied

`Onlyfied` is a meta crossover strategy game for this repo.

## Concept
- Pulls source inspiration from `Projects/data/games.json`
- Remixes ideas from the existing Edwin Estro games into one playable loop
- Balances `hype`, `clarity`, `trust`, `momentum`, and `burnout`

## Controls
- `1` — Amplify the hook
- `2` — Prototype the idea
- `3` — Stabilize the build
- `4` — Recover and refocus
- `N` — Start a fresh run

## Win condition
Reach `120` score with `trust` and `clarity` above `45` before day `14`.

The game is static and build-free by design.
